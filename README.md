# Teens With Vision Landing Site

A mobile-first one-page landing website for Teens With Vision.

## What to replace before publishing

1. `script.js`
   - Replace `WHATSAPP_NUMBER = '27000000000'` with the official WhatsApp number in international format, for example `27821234567`.

2. `index.html`
   - Replace the donation bank placeholders inside the `#bank-details` section.
   - Replace the three image placeholder blocks with real photos when available.
   - Replace partner slots in the `#partners` section with partner logos.

## Files

- `index.html` — site structure and content
- `styles.css` — mobile-first responsive styling
- `script.js` — mobile menu, scroll reveal, button ripple and contact actions
